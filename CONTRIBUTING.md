# Contributing to hasterolls5e

Code and localization (i.e. translations) contributions are accepted. You can also submit a feature request by submitting an issue on the issue tracker.

All contributions must follow [Foundry VTT's AI Content Policy](https://foundryvtt.com/article/ai-policy/) since this module is on their official package list. Pay special attention to section 6.6 since I intend to add this module to the "Zero AI" category.
